- # 考试内容与技巧
	- ## Part1：只是热身
		- 时间：4-5min
		- 内容：10-11个问题
			- 3个intro
			- 4个某topic
			- 4个某topic
		- 关注点：
			- Simon得分要点：流利，不要管语法和单词，想到什么说什么。不要有嗯和啊
			  logseq.order-list-type:: number
			- 不要管内容，考官不会判断你的内容是否正确
			  logseq.order-list-type:: number
		- 技巧：
			- 内容：直接回答问题+一个原因，如果时间不够就再补个例子
			- 得是完整的句子，不能只是单词和短语
	- ## Part2
		- 时间：3-4min
		- 内容：
			- 1分钟准备
			- 2分钟pre
		- 技巧：
			- 跟着卡上的内容来组织 pre 结构
			- Simon得分要点：主要关注的是 内容 和 **单词**，语法和连接词不是那么的重要
			- 录下来口语，转文字，检查自己的词汇和语法，以及故事叙事
			- 结构：
				- 开篇：
				  logseq.order-list-type:: number
				- 主题段落：跟着题卡的结构去讲就行
				  logseq.order-list-type:: number
				- 结尾：
				  logseq.order-list-type:: number
		- 素材准备内容：
			- 每一个大类准备两到三个theme，例如人物类的，theme为工作专注、乐于助人
			- 围绕这个theme去准备相应的词汇和句子
			- 对大类里的具体对象，比如明星、朋友，准备一个和theme相关的例子
