				- 上面两段中挑一个加入指令关键字
				  logseq.order-list-type:: number
		- 主体段：
			- 段内常见结构：Position + Reason + Example + Conclusion
				- Position（观点）：表明该段的中心思想。
				- Reason（理由）：解释为什么这个观点成立。
				- Example（例证）：提供具体的案例支持观点，最好是具体的、专有名词的案例，增强说服力。
				- Conclusion（总结）：总结该段的论点，承接下一段。
			- 段间：部分认可+ 我方观点
			- 模版：
				- [（to begin with, I agree to a certain extent that + 部分认可的对方观点） + （the reason is that + 理由 ）+ （For instance + 支持的例证）+（Thus + 重复部分认可的观点）]+[（However/Nevertheless, + if + 被否定的极端部分 + there will be + ）+（the reason is that + 理由 ）+ （For instance + 支持的例证）+（Thus + 自己支持的观点）]

			- 重述两个主旨段的3s版本
			- 模版：
